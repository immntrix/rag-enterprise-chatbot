# RAG Enterprise Chatbot

This project implements Retrieval Augmented Generation (RAG) to answer questions from uploaded PDFs using embeddings, FAISS vector database, and LLM.

## Features
- Upload PDF
- Convert text into embeddings
- Store in FAISS vector DB
- Ask questions from document
- Accurate, document-based answers

## Tech Stack
LangChain, OpenAI, FAISS, Streamlit, PyPDF

## Run
streamlit run app.py
